Game Overview

Project Gunning Wild is a 2D rail shooter set in the Wild West days.
Whilst roaming the desert towns, you'll come across a variety of different enemies.
The core of this game will be flair and speed, abiding to these core aspects will help you progress through the levels.

Influences

Operation Wolf

Operation Wolf is a light gun shooter arcade game which has won Game of the Year back in 1988. This played as an influence in the gameplay


Blood Bros

Blood bros. is a 90s shooting gallery arcade game, which has an art style that we would like to replicate into our own game
